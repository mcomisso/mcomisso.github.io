## FAQ

###### Creating a post? 

Create file as `_posts/YEAR-MONTH-DAY-title.MARKUP`.

###### Starting a draft?

Create file as `_drafts/YEAR-MONTH-DAY-title.MARKUP`.

###### Publishing?

Move from _drafts to _posts and `add, commit & push`.
